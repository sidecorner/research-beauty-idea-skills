# スキンケア・美容系ニッチアプリアイデア調査レポート
**調査年度: 2024年** | **対象: 30代以上の女性・敏感肌・ロゼーシャを持つ人々**
**方法: WebSearch（スキルなし・ベースライン）**

---

## ターゲット層の主要ペインポイント

### 30代以上の女性
- エストロゲン低下による乾燥・敏感化（更年期前後5年でコラーゲン最大30%減）
- ホルモン変動による大人ニキビの再発
- 60%が更年期前後でスキンケアルーティンを変更、48%がコスト増を経験

### ロゼーシャ（酒さ）患者
- 個人ごとに異なるトリガーの特定が困難
- 90%以上がトリガー管理で症状改善を報告するも体系的追跡ツールが少ない
- 約70%が自己肯定感低下・社会不安・うつを経験

### 敏感肌全般
- 15,000以上の化粧品成分の安全性判断が困難
- 製品の重ね付け順序・成分相性チェックが複雑

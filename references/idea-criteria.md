# What Makes a Good Idea

## Core Criteria

An idea is worth pursuing if it meets **all three**:

1. **Repeatedly mentioned** — the pain appears across multiple posts or threads, not a one-off complaint
2. **Clear user action** — there's an obvious thing to build: find, track, compare, learn, book, etc.
3. **Small-team scope** — achievable by 1–3 people (see `scoring-rubric.md` dimension 5)

## Cross-Platform Signal

Signal strength, ranked:

| Signal | Strength |
|--------|----------|
| Reddit + HN + Qiita all mention it | Strongest — broad, cross-cultural validation |
| Two of three sources | Strong |
| English only (Reddit + HN) | Good for global market |
| Qiita only | Japan-market specific opportunity |
| Single subreddit, one thread | Weak — do not prioritize |

Ideas appearing in all three sources — especially with both English and Japanese interest — are the strongest candidates for Japan-market-focused apps.

## Validation Signals to Look For

- "I wish someone made X" / "Why doesn't any app do Y" / "I've tried everything"
- A pain mentioned independently in 3+ subreddits
- Long comment threads (50+) with nuanced frustration
- Existing workarounds (spreadsheets, manual tracking) signal willingness to pay

## Competitive Risk Flags

- Flag ideas that overlap with well-funded startups (note the competitor and its funding stage)
- Score down if large incumbents (L'Oréal, Sephora, Amazon, Ulta) already own the space
- Niche demographics (mature skin, textured hair, sensitive skin, pregnancy-safe) are often
  underserved even when the parent category is crowded

## Quality Notes

- Prefer ideas with evidence from **multiple sources** — cross-platform signal is stronger
- When quoting user posts, paraphrase to avoid privacy concerns
- A score of 3 on any dimension is not a failure — it's useful calibration
- If fetch scripts return sparse data, note it in the report and suggest manual follow-up
  using the subreddits in `beauty-subreddits.md` or queries from `source-guidelines.md`
